# Clicker Starter Kit 🚀

A ready-made solution for launching a Telegram Mini App (TWA) in the “Tap-to-Earn” genre. 
This is a framework that can be quickly adapted to any theme (clicker, game, service).

### 📱 What it looks like


<img src="Game.gif" width="400">


### 🛠 Technology stack
- **Frontend:** React + Vite
- **Styles:** Tailwind CSS
- **Platform:** Telegram Mini Apps

### 💡 Core Features
- Energy System
- Coin Balance
- Upgrade Store
- Job Changes

### 🚀 How to Run the Project
1. Clone the repository: `git clone https://github.com/твое-имя/Clicker-Starter-Kit.git`
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
